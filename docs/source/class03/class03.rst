Automate your App Deployments with ACC and AS3
============================================

Explore F5 Application Services v3 (AS3) and the AS3 Configuration Converter (ACC) using a declarative API model to create applications and convert your existing BIG-IP configuration. During this hands-on lab you will learn the following:

•	How to use AS3 as a declarative endpoint for Virtual Server configuration. 

•	How to convert an existing configuration to AS3 JSON declaration.

Expected time to complete: **30 minutes**
